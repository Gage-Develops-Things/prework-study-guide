HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The "ul" - "unordered list" - element is known as a parent element, and it can contain as many "li" - list item - elements as needed.
  • The "ul" element creates an unordered list, or a bulleted list, in which more content can be placed.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • An external CSS file needs to be linked to the HTML file via a "link" element in the head.
  • CSS styles are defined with property-value pairs.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git commit: commits the code you've been working on.
  • git -m leaves a message on the file you are about to commit.

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Arrays can hold multiple related data points.
  • Functions must be called in order for the control flow to process them.
